# Minecraft Mods

## How To

https://codakid.com/guide-to-minecraft-modding-with-java/

Run this to get ready for Eclipse:

`./gradlew genEclipseRuns`

Build jar with 

`./gradlew build`

Find jar under `/build/libs`

Move jar under `%appdata#/.minecraft/mods`
